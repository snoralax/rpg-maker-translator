RPG Maker Translator
===============================

version number: 0.0.1
author: Snoralax

Overview
--------

A translation tool for RPG Maker (MV) games.

Installation / Usage
--------------------

To install use pip:

    $ pip install rpg-maker-translator


Or clone the repo:

    $ git clone https://github.com/snoralax/rpg-maker-translator.git
    $ python setup.py install
